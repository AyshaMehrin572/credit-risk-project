# Credit Risk Prediction using Logistic Regression

This project is developed to predict the creditworthiness of customers using Machine Learning. Logistic Regression is used to classify whether a customer is likely to default on a loan based on financial and demographic data.

---

## Project Objective

The main objective of this project is to build an automated credit risk prediction system that helps banks and financial institutions make better loan approval decisions.
